How It Works


Upload

This page here is to upload audio files to be processed for transcription. All successful uploads will be shown on the "Audio List" page.


Audio List

This page consists of all uploaded audio files. The audio files can be transcribed accordingly and the results will be shown. All audio files that are flagged with unwanted words will be sent to the "Assessment" page.


Assessment

This page shows all audio files that have been flagged with unwanted words and are available for assessment. The assessor can listen to the audio file and decide on whether it has been flagged correctly.


About

This page allows you to modify certain settings/parameters for databases. For security reasons, not all settings can be changed. Users have the opportunity to modify the list of unwanted words when transcriptions are being processed.
